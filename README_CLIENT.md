# دليل تشغيل مشروع متجر شركة الشفاء لتوزيع الأدوية

أهلاً بك. هذا دليل مبسط لإدارة وتشغيل متجرك الإلكتروني.

## 1. الوصول إلى المتجر
يمكنك الدخول إلى المتجر من خلال الرابط التالي:
[رابط المتجر](https://ais-dev-7b2mwlkw3ptyq66zedhhqs-179686310005.europe-west2.run.app)

## 2. تسجيل الدخول كمدير (Admin)
للوصول إلى لوحة التحكم الخاصة بالإدارة، يرجى اتباع الخطوات التالية:
1. افتح رابط المتجر في المتصفح.
2. اضغط على زر تسجيل الدخول واستخدم بريدك الإلكتروني (Gmail).
3. بعد تسجيل الدخول، ستحصل على صلاحيات "مدير" تلقائياً بمجرد تعيين دورك من قاعدة البيانات.

## 3. محتويات لوحة التحكم
من خلال لوحة التحكم يمكنك:
- **إدارة المنتجات:** إضافة، تعديل، أو حذف الأدوية والمنتجات.
- **إدارة الطلبات:** متابعة طلبات العملاء وتغيير حالاتها (قيد المعالجة، تم الشحن، إلخ).
- **إدارة الحجوزات:** الاطلاع على الحجوزات القادمة وإدارتها.
- **إدارة الإعدادات:** تعديل بيانات شركة الشفاء (العنوان، الهاتف، حسابات الدفع، إلخ).

## 4. مساعدة ودعم
لأي استفسارات تقنية حول تشغيل المتجر أو إطاره البرمجي، يمكنك التواصل مع المطور.
